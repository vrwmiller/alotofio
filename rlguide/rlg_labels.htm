Record Label Guide Research Page

This page will provide access to all of the labels currently being researched for the 2nd edition of the Record Label Guide.  As we progress in the research of a label, we will add a .PDF file of the variations we have noted. Once the research is done, the old links will be removed. Click on any of the hyperlinked labels and a PDF file will open in a separate window.  You must have Adobe Acrobat Reader installed to view the PDF file.

If the label name is not hyper linked, we have either 1) completed research, 2) haven't created the PDF file yet, or 3) haven't started the research.  Is there a label you would like to see included but it is not shown below?  Just send us an email and let us know.  It would also help if you could provide some initial research information on it.

Comments, suggestions, compliments, and criticism are solicited.  Please send them via email.  Thanks
